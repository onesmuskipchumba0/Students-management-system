# School Management Software

Welcome to the School Management Software repository! This desktop application is built using Python with PyQt5 for the graphical user interface and SQLAlchemy for database management. The system aims to streamline school administrative tasks by providing features for managing students, teachers, courses, and grades.

## Features

- **User-Friendly Interface**: The application offers an intuitive and easy-to-navigate graphical interface for managing school-related data.

- **Database Integration with SQLAlchemy**: Utilizes SQLAlchemy for efficient management and retrieval of data, ensuring a seamless experience.

- **Student and Teacher Management**: Add, edit, and delete student and teacher profiles with ease.

- **Library Management**: Manage school libraries and assign books to students.

## Prerequisites

Make sure you have the following installed before running the application:

- Python 3.x
- PyQt5
- SQLAlchemy

You can install the required dependencies using the following command:

```bash
pip install PyQt5 SQLAlchemy
```

## Getting Started

1. Clone the repository to your local machine:

```bash
git clone https://github.com/ONESMUSBETT/Students-management-system.git
```

2. Navigate to the project directory:

```bash
cd school-management-software
```

3. Run the application:

```bash
python main.py
```

## Usage

- Use the graphical interface to manage student and teacher profiles.
- Add, edit, and delete courses as needed.
- Record and update student grades for each course.
